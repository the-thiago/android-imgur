# Imgur Cat Images

### About
The app is a cat gallery built using the Imgur API.

### Technologies

- MVVM architecture
- Clean Architecture
- Multi-Module
- Jetpack Compose
- Dagger Hilt
- Retrofit
- Room
- Data cache with Paging3 and Remote Mediator
- Local unit tests with Mockito
- Instrumented tests with Compose
- Detekt
